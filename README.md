###Komendy:
- `**/chat <on/off/cc>**` - Manipuluje chatem. Wlacza/Wylacza/Czysci
- `**/broadcast <tekst>**` - Wysyla globalna wiadomosc. 
- `**/changelore <tekst>**` - Zmienia opis przedmiotu.
- `**/changename <tekst>**` - Zmienia nazwe przedmiotu.
